About Weather Helper

Weather Helper is a web application designed to help users quickly understand weather conditions and make better decisions for their day. Users can search cities, receive current weather information, and get simple weather recommendations based on temperature, wind, and conditions.

Weather Helper focuses more on presenting weather data rather than providing large amounts of information at once so that the user doesn't feel overwhelmed. The application has a more cleaner and user-friendly way while still being able to give practical advice, such as whether if the user should bring an umbrella or dress warmly.

This project uses OpenWeather API to get live weather data and Supabase as a backend database for saving locations. The application also uses Chart.js to display weather informaiton visually and Leaflet.js to display maps for searched cities.

Target Users

  • People who plan on doing outdoor activities or travel
  • Commuting to work or school
  • Users who want a quick and simple weather advice
  • Anyone looking for an easier weather experience

Language/Technologies Used
